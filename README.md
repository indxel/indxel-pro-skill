# indxel-pro-skill

Claude Code skill for **[indxel](https://indxel.com)** Pro — advanced SEO infrastructure.

## What it does

This skill teaches Claude Code how to use indxel's advanced features for live site analysis:

- **Crawling** — crawl live sites, analyze scores across pages, detect broken links/images
- **Keyword research** — extract keywords, analyze competition, find content gaps
- **Indexation** — submit URLs to Google (Indexing API + IndexNow), check index status
- **Performance** — PageSpeed analysis, Core Web Vitals, asset verification
- **Sitemap & robots.txt** — parse, compare, validate against actual pages

## Install

```bash
claude install-skill indxel/indxel-pro-skill
```

## Trigger phrases

The skill activates when you mention:
- "crawl site", "keyword research", "indexation", "Google index"
- "submit URL", "site performance", "PageSpeed", "content gaps"
- "link analysis", "connect dashboard"
